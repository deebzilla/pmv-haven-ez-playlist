# Privacy Policy

PMV Haven Playlist Helper does not run its own server and does not collect, store, sell, or transmit personal data to any third-party service controlled by the developer.

## What the extension accesses

- `storage`
  - Used only to save the selected PMV Haven playlist ID or URL and the menu label in the browser locally.
- `tabs`
  - Used only to find currently open PMV Haven video tabs when the user chooses the bulk-add action.
- `contextMenus`
  - Used only to provide right-click actions.
- `scripting`
  - Used only to run the playlist-add request on a PMV Haven page in the user's own browser session.
- `notifications`
  - Used only to show success or error messages to the user.

## What data leaves the browser

When the user triggers an add action, the extension sends requests only to PMV Haven endpoints that are already used by the PMV Haven website, using the user's existing logged-in browser session.

The extension does not send data to any analytics platform, developer server, or advertising service.

## Data retention

The extension stores only:

- the configured playlist URL or ID
- the chosen menu label

This data remains in the browser's local extension storage until the user changes it or removes the extension.
