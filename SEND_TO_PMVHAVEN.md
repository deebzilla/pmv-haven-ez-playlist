# Message To PMV Haven

Hi PMV Haven team,

I built a small browser extension called **PMV Haven Playlist Helper** to solve a friction point in playlist building.

## The problem

Right now, if someone has a bunch of PMV Haven videos open or wants to save multiple finds from browse/search pages, they have to open each video and add it individually.

## What this extension does

- right-click a PMV Haven video link and add it straight to a chosen playlist
- right-click the current PMV Haven video page and add it
- bulk-add all open PMV Haven video tabs with one click from the extension popup

## Why it is useful

It cuts down the number of clicks a lot for playlist curating and makes power-user workflows much smoother.

## Technical note

The extension uses PMV Haven's existing playlist endpoints from the user's logged-in browser session. It does not scrape UI clicks and does not use a third-party backend.

## If helpful

I'm happy for you to:

- share the extension as-is
- adapt the idea into PMV Haven directly
- or use the code as a starting point for a native bulk-add feature

If you'd like, I can also help turn this into a built-in site feature instead of an extension.

Thanks.
